# 图审云平台 - 系统说明文档

## 项目结构

```
tushen-system/
├── backend/              # Python FastAPI 后端
│   ├── main.py          # 主程序入口（所有API）
│   ├── models.py        # 数据库模型
│   ├── auth.py          # 认证逻辑
│   ├── requirements.txt # Python依赖
│   ├── uploads/         # 文件上传目录（自动创建）
│   └── tushen.db        # SQLite数据库（自动创建）
├── frontend/
│   └── index.html       # 前端单页应用
└── 启动系统.bat          # 一键启动脚本
```

## 快速启动

### 方式一：双击 `启动系统.bat`（推荐）
自动安装依赖 → 启动后端 → 打开前端

### 方式二：手动启动
```bash
# 1. 安装依赖
cd backend
pip install -r requirements.txt

# 2. 启动后端
python main.py

# 3. 用浏览器打开 frontend/index.html
```

## 功能模块

### 用户与认证
- 手机号/邮箱注册
- 用户类型：材料商 / 设备商 / 设计院 / 设计师
- 实名认证与资质上传（营业执照/执业证）
- 管理员人工审核（待审核/通过/驳回）

### 需求发布（甲方）
- 新建/编辑/删除需求（支持草稿）
- 上传图纸文件（DWG/PDF/BIM/ZIP）
- 设置预算与付款方式（一次性/分阶段）
- 发布需求、需求状态管理
- 查看并选择中标报价方
- **选择中标后自动创建订单**

### 甲/乙方角色区分
- 用户注册时可选择：甲方（业主/建设单位）或乙方（设计院/设计师/材料商/设备商）
- 甲方专属：我的需求、发布需求
- 乙方专属：我的报价、接单大厅
- 双方共有：订单管理、图纸管理、纠纷处理

### 接单与报价（乙方）
- 浏览需求大厅，支持专业/金额/关键词筛选
- 提交/编辑/取消报价
- 查看我的报价记录和中标状态

### 交易与资金托管
- 创建订单（选定设计方后）
- 订单支付（资金托管至平台）
- 买方确认验收 → 平台放款
- 订单状态：待付款 → 进行中 → 已完成

### 图纸管理
- 设计方上传图纸文件（含版本号）
- 双方均可下载查看
- 需方可添加修改意见

### 通知系统
- 站内消息通知（系统消息/订单进度）
- 未读消息角标提示
- 点击标记已读

### 纠纷处理
- 用户发起纠纷（关联订单）
- 上传证据文件
- 客服后台介入处理
- 纠纷结果：退款/放款

### 投诉反馈
- 用户提交投诉/建议
- 后台可回复处理
- 投诉结果通知用户

### 审核中心（审核员/客服）
- 用户资质审核（通过/拒绝）
- 纠纷处理与裁决
- 投诉反馈管理与回复

### 后台管理（管理员）
- 用户审核管理（通过/驳回）
- 需求/订单全局管理
- 纠纷处理与裁决
- 数据统计面板

## API 文档
启动后访问：http://localhost:8000/docs

## 技术栈
- **前端**：HTML5 + CSS3 + Vanilla JS（SPA）
- **后端**：Python FastAPI + SQLAlchemy + SQLite
- **认证**：JWT Token（jose）+ bcrypt密码加密
- **文件**：本地存储（uploads/目录）
